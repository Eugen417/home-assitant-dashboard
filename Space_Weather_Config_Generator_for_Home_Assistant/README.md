# 🛠 Space Weather Config Generator for Home Assistant

Умный веб-генератор, который в один клик создает готовую конфигурацию (YAML, скрипты, промты) для отслеживания магнитных бурь и полярных сияний в вашем Home Assistant. Данные берутся от [ИКИ РАН](https://iki.rssi.ru/).

🌍 **Поддерживает более 100 городов России и ближнего зарубежья!**

[Здесь можно добавить скриншот интерфейса генератора, например, `![generator_preview](link_to_image)`]

## ⚡️ Как это работает?

Вам не нужно вручную прописывать ссылки или ID городов. Просто:

1. Откройте файл `ha_generator_full_v8.html` в любом браузере.
2. Выберите ваш город из выпадающего списка.
3. Генератор моментально подставит нужные префиксы и ID во все блоки кода.
4. Скопируйте нужный блок (кнопка «Копировать») или скачайте всё одним `.txt` файлом.

## 🧰 Что генерирует этот инструмент?

Генератор создает 6 готовых блоков кода, адаптированных под выбранный вами город:

### 1. `aurora_sensors.yaml` (Датчики)

Готовый код для вашего `configuration.yaml` (REST и Scrape сенсоры).
*(⚠️ Внимание: Если вы используете [официальную HACS-интеграцию](https://github.com/Eugen417/HA_xras_space_weather), этот блок вам **не нужен** — интеграция создаст сенсоры автоматически).*

### 2. `dashboard_card.yaml` (Информационная карточка)

Код для дашборда (Vertical Stack). Включает в себя видео-глобус, графики активности и красивую текстовую сводку (Markdown) с автоматическим расчетом угрозы (от "Спокойно" до "Экстремальная буря G5").

### 3. `yandex_script.yaml` (Голосовая сводка Алисы)

Готовый скрипт для интеграции Яндекс Станции (`yandex_station`).
Добавьте этот скрипт, и Алиса сможет зачитывать вам утреннюю сводку:

> *"Доброе утро! Краткая сводка для города Москва. Прямо сейчас слабая буря, текущий индекс 5. Максимальный ожидаемый уровень за сегодня — 6. Вероятность полярного сияния 12%."*

### 4. `aurora.html` (Видео-фон)

Простой HTML-файл для папки `www`, который подтягивает актуальную анимацию глобуса с зонами полярных сияний от ИКИ РАН.

### 5. Инструкция по установке

Краткая памятка о том, куда вставлять каждый из сгенерированных блоков в интерфейсе Home Assistant.

### 6. Системный Промт (Для локальных ИИ)

Текстовый промт для обучения локальных AI-ассистентов внутри Home Assistant. Промт "объясняет" нейросети, к каким датчикам обращаться, чтобы правильно отвечать на вопросы пользователя о солнечных вспышках, Kp-индексе и полярных сияниях.

## 🚀 Полноценная интеграция (HACS)

Если вы не хотите возиться с YAML-кодом датчиков, обратите внимание на нашу **[Пользовательскую HACS Интеграцию](https://github.com/Eugen417/HA_xras_space_weather)**.
Она добавляет все сенсоры в 2 клика через графический интерфейс Home Assistant и включает в себя эксклюзивную карточку с динамическими видео-анимациями магнитных бурь.
