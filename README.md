# Home Assistant Dashboard Generators

A collection of scripts for generating YAML configurations.

## 🇷🇺 Русский

### Space Weather Config Generator
* **Ссылка:** https://github.com/Eugen417/home-assitant-dashboard/tree/main/Space_Weather_Config_Generator_for_Home_Assistant
* **Описание:** Генерирует YAML-конфигурацию для сенсоров космической погоды и соответствующих карточек интерфейса.
* **Примечание:** Также существует отдельная интеграция с продвинутой анимированной карточкой (не входит в состав генератора): https://github.com/Eugen417/HA_xras_space_weather

### Miele Washer Card Generator
* **Ссылка:** https://github.com/Eugen417/home-assitant-dashboard/tree/main/miele_washer_card_generator
* **Описание:** Генерирует YAML-код карточки для отображения статуса, времени цикла и рабочих данных стиральных машин Miele.

## 🇺🇸 English

### Space Weather Config Generator
* **Link:** https://github.com/Eugen417/home-assitant-dashboard/tree/main/Space_Weather_Config_Generator_for_Home_Assistant
* **Description:** Generates YAML configuration for space weather sensors and dashboard cards.
* **Note:** An integration with an advanced animated card is also available (not part of the generator itself): https://github.com/Eugen417/HA_xras_space_weather

### Miele Washer Card Generator
* **Link:** https://github.com/Eugen417/home-assitant-dashboard/tree/main/miele_washer_card_generator
* **Description:** Generates YAML code for a Home Assistant card displaying Miele washing machine states, cycle times, and operational data.
